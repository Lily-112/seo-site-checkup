## Introduction

This project performs automated SEO analysis by running dozens of industry-standard tests related to on-page optimization, site structure, mobile usability, security, and performance.
It solves the problem of manual auditing by offering a repeatable, automated, and scalable way to evaluate website health.
Itâ€™s ideal for SEO specialists, web developers, digital agencies, and performance engineers.

### Why Automated SEO Checking Matters

- Identifies critical issues that impact search rankings.
- Speeds up repetitive audit workflows.
- Ensures consistent evaluation across different sites.
- Provides structured data suitable for reporting or dashboards.

## Features

| Feature | Description |
|--------|-------------|
| Meta Tag Checks | Evaluates meta titles, descriptions, keywords, and social metadata. |
| Content & Keyword Analysis | Extracts keyword cloud, usage, and common phrase patterns. |
| Technical SEO Tests | Runs robots.txt, sitemap, canonical, redirects, and indexability checks. |
| Performance Audit | Measures HTML size, DOM size, loading speed, caching, and compression. |
| Mobile Usability | Tests viewport, responsive layout, media queries, and mobile snapshots. |
| Security Validation | Checks HTTPS, SSL, HSTS, mixed content, and safe browsing status. |
| Image Optimization | Analyzes alt tags, aspect ratio, compression, caching, and modern formats. |
| Code Quality Checks | Detects inline CSS, deprecated tags, JS errors, and console issues. |
| Server & Network Tests | Evaluates server signatures, directory browsing, CDN usage, and HTTP2. |
| Structured Data & UX | Validates schema markup, CLS, LCP, and custom error page presence. |

## What Data This Scraper Extracts

| Field Name | Field Description |
|------------|------------------|

| metaDescription | Evaluated meta description and quality score. |
| keywordAnalysis | Keywords, density, cloud, and usage metrics. |
| socialTags | OpenGraph and Twitter metadata fields. |
| backlinks | Basic backlink references and domain checks. |
| speedMetrics | Load time, LCP, CLS, and JS execution results. |
| imageChecks | Alt text, caching, compression, and format insights. |
| securityResults | SSL, HTTPS, HSTS, mixed content, and safe browsing. |
| mobileUsability | Mobile responsiveness and viewport validation. |
| structuredData | Schema markup presence and parsing results. |
